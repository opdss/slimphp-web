Slim-Skeleton

Slim 框架 3.X 应用程序骨架

使用这个应用程序骨架，可以快速配置并开始开发新 Slim 3 应用程序 。此应用程序使用了最新的 Slim 3 以及 PHP-View 模板渲染程序。它还使用了 Monolog 作为日志工具。

本应用程序骨架使用 Composer 进行构建。这使得配置全新 Slim 框架应用程序更加快速简单。

安装应用程序

在你想要新建 Slim 应用程序的目录中，运行以下命令：

php composer.phar create-project slim/slim-skeleton [my-app-name]

用你的应用程序所需的目录名称来替换 [my-app-name] 。接下来：

  • 将虚拟主机的根目录指向你的应用程序的 public/ 目录。
  • 确保 logs/ 目录是 web 可写入的。

搞定！开始折腾你的炫酷应用吧。
